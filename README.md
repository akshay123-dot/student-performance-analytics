# 🎓 Student Performance Analytics System

## Project Overview

The Student Performance Analytics System is a Python-based application developed using Streamlit, Pandas, and Matplotlib. The system helps educational institutes manage student records, analyze academic performance, identify weak students, and generate reports from student marks data.

This project was developed as part of the TEYZIX Core Internship Program (June 2026).

---

## Features

### Student Record Management

* Add new student records
* Update student marks
* Delete student records
* View all student records

### Performance Analysis

* Calculate total marks
* Calculate average marks
* Calculate percentage
* Assign grades (A, B, C, Fail)

### Analytics Dashboard

* Top-performing students
* Weak students list
* Subject-wise average marks
* Class-wise performance comparison

### Reporting System

* Export performance reports in CSV format
* Maintain persistent student records

### Visualization

* Subject-wise average marks graph
* Interactive Streamlit dashboard

## Installation

### 1. Clone or Download the Project

```bash
git clone <repository-url>
```

### 2. Install Required Packages

```bash
pip install -r requirements.txt
```

### 3. Run Application

```bash
streamlit run app.py
```
